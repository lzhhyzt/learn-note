一，安装git 1. 安装git软件 国内镜像：https://pan.baidu.com/s/1kU5OCOB#list/path=%2Fpub%2Fgit 国外镜像：https://git-for-windows.github.io/ 2. 安装小乌龟和小乌龟汉化包，使得git文件有状态标识 (例如已提交是绿色图标) 右键菜单git小乌龟设置中 -> 图标覆盖- > 外壳增强 3. 设置全局的设置 git config --global user.name "wenye" git config --global user.email "1583247975@qq.com" 二，设置git bash的一些语言配置 1.在git的安装目录etc\git-completion.bash中 加上alias ls='ls --show-control-chars --color=auto' 使得在 Git Bash 中输入 ls 命令，可以正常显示中文文件名。 2.在git的安装目录etc\inputrch中 设置 set output-meta on set convert-meta off 使得在 Git Bash 中可以正常输入中文 3.在git的安装目录etc\profile中 加上 export LESSCHARSET=utf-8 使得 $ git log 可以正常显示中文 4.在git的安装目录etc\gitconfige中 加上 [gui] encoding = utf-8 使得git gui 中正常显示代码中的中文。 [i18n] commitencoding = GB2312 让commit的为utf8 三，安装git服务器 看我自己写的linux笔记
